# Clínica Vida & Beleza

Site institucional desenvolvido em HTML para apresentar a Clínica Vida & Beleza, seus serviços, horários de atendimento e informações de contato.

O projeto foi criado com o objetivo de praticar a estruturação de páginas web utilizando HTML, navegação entre páginas e organização de conteúdo para um site simples e funcional.

## 📌 Sobre o projeto

A **Clínica Vida & Beleza** é um site simples voltado para apresentação de uma clínica especializada em estética, saúde e bem-estar.

O site possui páginas separadas para facilitar a navegação do usuário e apresentar as principais informações da clínica de forma clara.

## 📄 Páginas do site

O projeto contém as seguintes páginas:

- **Página Principal**: apresentação inicial da clínica
- **Sobre a Clínica**: informações sobre a história e estrutura da clínica
- **Horário de Atendimento**: tabela com serviços e horários disponíveis
- **Contato**: telefone, WhatsApp, endereço e formulário de contato
